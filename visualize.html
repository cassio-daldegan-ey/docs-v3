class visualize.Visualization(env_path='env.json', flag_cloud: bool = False)[source]

Classe para processar e visualizar os dados.

A classe possibilita combinar os DataFrames de dados reais e preditos, gerar a média móvel por grupo e salvar os resultados em arquivos csv.
